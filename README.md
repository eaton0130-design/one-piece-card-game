# 航海王卡牌冒險遊戲

歡迎來到《航海王卡牌冒險遊戲》！
這是一款以《ONE PIECE》世界觀為背景的文字冒險遊戲。你將扮演魯夫，展開一段充滿戰鬥、劇情與抽卡樂趣的海上冒險。

## 遊戲目標
- 觀看精彩劇情
- 與敵人進行戰鬥
- 抽取新角色加入隊伍
- 賺取貝里，提升你的冒險實力

## 遊戲方式
進入遊戲後，你可以選擇：
1. 觀看劇情模式
2. 進行自由戰鬥
3. 進行角色抽卡
4. 離開遊戲

## 你會看到的內容
- 熱血的航海王劇情片段
- 角色技能與戰鬥畫面
- 互動式小遊戲
- 抽卡獲得新夥伴

## 戰鬥說明
戰鬥是回合制的：
- 你可以選擇攻擊或防禦
- 每次攻擊會使用角色技能
- 有些技能會造成特殊效果
- 勝利後可以獲得貝里獎勵

## 抽卡說明
- 每小時有一次免費抽卡機會
- 也可以用貝里直接抽卡
- 抽到重複角色時，會得到補償金

## 遊戲小提醒
- 先完成劇情，會更容易了解遊戲世界
- 多戰鬥可以賺取貝里
- 抽卡可以讓你的隊伍變得更強大

## 開始遊戲
直接執行主程式即可開始冒險。
如果你喜歡《ONE PIECE》與卡牌戰鬥，這款遊戲會很適合你！
